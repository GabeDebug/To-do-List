# To-Do List em C# (Console)

Projeto simples de **Lista de Tarefas (To-Do List)** desenvolvido em **C#**, utilizando aplicação de **console** e persistência de dados em **arquivo `.txt`**.

Esse projeto é ideal para quem está aprendendo os fundamentos da linguagem C# e conceitos básicos como listas, classes, arquivos e lógica de programação.
